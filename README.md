# Dark Mode Toggle

## Overview
A simple static web application demonstrating a Dark/Light mode toggle using vanilla HTML, CSS, and JavaScript.

## Features
- Toggle between Dark and Light themes with a button click.
- DOM manipulation examples (create elements, event listeners, class toggles).
- Additional JS experiments (arrays, loops, map/filter/reduce, async fetch).

## Files
- `index.html`: Basic structure with heading, paragraph, div, and toggle button.
- `style.css`: Styles for `.dark` (black bg, white text) and `.white` (white bg, black text) modes.
- `index.js`: Core logic for mode toggle and DOM updates.

## How to Run
1. Open `index.html` in any web browser.
2. Click **Change Mode** to toggle themes.
3. Inspect console for JS outputs.

No build tools or dependencies required – fully self-contained.

## Demo
- Initial: Light mode.
- Toggle: Switches to Dark mode and back.
